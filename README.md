# NAME

CPAN::Test::Dummy::Perl5::VersionQV - qv() version

# SYNOPSIS

    use CPAN::Test::Dummy::Perl5::VersionQV;

# DESCRIPTION

CPAN::Test::Dummy::Perl5::VersionQV is a dummy CPAN distribution with the use of `qv()` in its VERSION line.
